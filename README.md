安居街周邊：行人路權受阻與交通違規實證分析專案

Anju Street Pedestrian Right-of-Way & Traffic Violation Evidence Dashboard

專案背景 (Project Background)

在安居街及其周邊路段，常態性的交通違規已嚴重壓縮行人的生存空間。其中最具毀滅性危害的，是車輛頻繁違規佔用路口之「人行道外推處 (Curb Extension)」、無障礙坡道與行人穿越道。

這種行為直接截斷了安全的步行路網，迫使輪椅使用者、推嬰兒車的家長、學童與一般行人，必須無奈地走上車道與汽機車爭道，導致險象環生。為打破「開單取締治標不治本」的無效循環，在地居民與通行權關注組發起本專案，透過實地影像採證、AI 數據鑑識與資料視覺化，將冰冷的違規照片轉化為具備強大說服力的市政安全報告。

核心訴求 (Core Demands)

本專案不以「增加罰款」或單純的「科技執法」為最終目的。我們透過數據庫推演證明，事後的追懲無法阻擋當下的物理危害。

我們強烈要求市府相關局處：

**立即於高風險路口與「人行道外推處」，全面重建『實體金屬防護柵欄與防撞桿』。**唯有透過不可跨越的物理防禦，才能從根本阻絕機動車輛入侵，徹底終結行人地獄，還給市民一條安全回家的路。

儀表板亮點功能 (Dashboard Features)

本專案包含一個無需後端伺服器、可部署於 GitHub Pages 的高互動視覺化儀表板 (index.html)：

威脅黑數推演模型 (Threat Extrapolation Model)

系統不只呈現「拍攝當下」的數據，更導入在地長期觀測經驗（如尖峰時段每小時 13 件），自動推算出令人震撼的真實日均違規量與全年潛在危害次數，揭露冰山底下的違規黑數。

高對比市政警示 UI (High-Contrast Civic UI)

採用智庫報告與戰情雷達風格，將「行人動線受阻」等致命危害以最大字體、最高對比色（玫瑰紅）獨立標示，為議會質詢與媒體報導提供最具視覺張力的畫面。

空間熱點地圖整合 (Spatial Mapping Integration)

無縫嵌入 Google My Maps (KML) 圖資，將所有採證影像精準標定於地圖上，直觀呈現「地獄路段排行榜」。

24H 時段與樣態分析 (Temporal & Structural Analysis)

內建 Chart.js 引擎，自動繪製違規發生的高峰時段折線圖，以及各項違規法條的結構佔比圓餅圖。
